Sobre Mim

Eu sou Marcelo Henrique e sou um desenvolvedor back-end com vasta experiência em projetos para FiveM usando Lua, JavaScript e C#. Trabalho elaborando soluções otimizadas, consistindo em desempenho, segurança e escalabilidade. Sou formado em Sistemas de Informação e possuo uma base sólida em desenvolvimento de jogos e web.

Experiências

Tenho mais de 5 anos de experiência em desenvolvimento web, com foco em aplicações modernas e integrações profundas com a plataforma FiveM. Domino diversas linguagens e frameworks, sempre buscando performance, segurança e estética.

Python

Scripts e automações eficientes para backend, análise de dados e integração com APIs. Utilizado para desenvolver ferramentas rápidas, bots e soluções robustas com foco em produtividade.

JavaScript

Criação de interfaces dinâmicas e interativas com Vue.js. Uso avançado de manipulação DOM, eventos, APIs REST, e lógica assíncrona com Promises e async/await.

Lua

Linguagem principal utilizada em scripts para servidores FiveM. Experiência sólida com client.lua, server.lua, e integração com front-end via NUI, otimizando lógica de jogo e desempenho.

C#

Desenvolvimento de aplicações desktop, scripts para servidores, e integração com bases de dados usando Entity Framework. Experiência com estruturas como MVC e lógica de negócio em back-end.

C++

Conhecimento aplicado em otimizações de desempenho e estruturas de sistemas, principalmente em bibliotecas nativas e extensões. Útil em contextos que exigem performance crítica.

HTML

Estrutura semântica de interfaces web modernas com foco em acessibilidade, SEO e organização clara do conteúdo. Uso avançado de componentes reutilizáveis e tags semânticas.

CSS

Estilização responsiva com uso de variáveis, animações, e designs modernos. Aplicação de conceitos como Flexbox, Grid e media queries para experiência fluida em todas resoluções.

SQL

Modelagem de dados, criação de estruturas relacionais otimizadas e desenvolvimento de queries eficientes. Experiência com MySQL, MariaDB e integração com back-ends em PHP, Node.js e C#.
